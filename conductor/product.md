# Product: InFocus

## Overview
InFocus is a professional Movie & TV Show Watchlist PWA (Progressive Web App) designed for movie enthusiasts who want a "Netflix-style" experience for tracking their personal cinema history and future watchlists.

## Core Features
- **Global Search:** Search for Movies and TV Shows via TMDB and OMDb (Fallback).
- **Watchlist Management:** Add titles to a personal collection, mark as watched, or favorite.
- **TV Show Support:** Detailed view for series including creators, first air dates, and media type indicators.
- **Gamification:** Achievement system and viewing statistics.
- **User Profiles:** Personalized profiles with display names, language settings, and data portability (Export/Import).
- **Security:** Strict data isolation via Supabase Row-Level Security (RLS).

## Target Audience
Movie buffs, series bingers, and collectors who value a clean, fast, and secure interface.
